# SystemCraft AI | Client 🎨

Frontend application for SystemCraft AI, built with React, Vite, and Tailwind-inspired Glassmorphism.

## 🚀 Key Features
- **Blueprint Dashboard**: Manage all your system architectures.
- **Architect View**: Real-time refactorings, schema generation, and API planning.
- **Zustand Store**: Centralized state management for projects and auth.
- **Monaco Editor**: High-performance code previews.

## 🛠️ Commands
- `npm run dev`: Start development server.
- `npm run build`: Build for production.
- `npm run preview`: Preview production build.

## ⚙️ Configuration
API endpoints are proxied via `vite.config.js` to `http://localhost:8080`.
Environment variables can be set in `.env`.
